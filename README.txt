# exoGit

Mise en pratique des notions acquises dans les deux premières parties du cours sur Git et Github.

Il s'agit d'un repository Git simple contenant :

-ce fichier nommé "README.txt" qui présente le projet;
-5 fichiers (2 requis min pour l'exercice);
-7 commits distincts montrant l’ajout et la modification des fichiers (4 requis min pour l'exercice).
